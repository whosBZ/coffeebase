# Cafe Directory

## What is it?

* Database of cafes from around the world with information about them
* Users can also review the cafes
* Users can also link beans to the cafe if they find they are available there
* Can also provide information on the equipment sold
* Can provide what types of coffee they make

## How is it handled

### New cafes

* Users can submit cafes for review by an admin/moderator and once approved will appear in the public directory.
* Moderators cannot bypass the review when submitting, it must always be reviewed by someone else.
* If the submission is bad or it already exists the user will be notified of its existence or asked to provide better information.
* User must provide the name, description and location using a exact GPS coordinate (Maybe use google maps api here).
* Cafes can request to update information on their location if they can prove they are the owners to override original submissions.

### Review Cafes

* For existing cafes users should be able to provide reviews on the cafe either in a short form or longer writeup in a different area.
* Reviews must not be useless information and will have an auto review feature to prevent profanity or harmful wording.

### Available Beans/Equipment

* There needs to be a global location for beans and equipment that can be linked to a cafe if available
* We don't have a way to track stock so it will just have to go on a "It has been bought here" 
* Users can flag items as no longer available at the locationa
